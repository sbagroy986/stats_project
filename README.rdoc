This is a Probability and Statistics (MTH201) course project at Indraprastha Institute of Information Technology, Delhi.
The goal of the project is to establish a relationship between the disposition of a person towards a song and their knowledge of the artist of the song. The effect of knowing an artist on a person's liking of the song.

You can find the website hosted here:  stats-project.herokuapp.com

The website is involved with the data collection portion of the project. A part of the statistical analysis can be found in /analysis , which mainly consists of a few graphs and scripts for creating the same. The calculations for Hypothesis Testing ha sbeen done locally.
